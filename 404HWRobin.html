HW1 SP500 and Treasury Bill Rate

For this homework assignment, we compute the average returns of SP500 (Including dividends) from 01/01/1973 to 01/01/2015 in different time horizon
In addition, we want to compute the excess rate of return of SP500, namely, return of SP500 minus return of Treasury Bill.

Full Body Text

  • I downloaded SP500 from Yahoo Finance in Daily Frequencies and used "Close" for all the computations, or we could download SP500 value-weighted return including dividends from wrds
  • The Panda function df["Close"].resample('').ffill().pct_change() allows me to compute returns in frequencies I wish
  • I downloaded Daily Series of Return Adjusted from wrds and used df["Close"].resample().ffill() to adjust frequencies
    • The lesson we learn from observing the annualized excess returns is the longer you put your money in SP500, the larger the returns are. This is possibly because SP500 grows in a long-term whereas risk-free rate declines in a long-term. Therefore the differences between two are expanded in long time horizon.

Executive Summary

HW2 SP500 Excess

Aside from the commonly used definition of "excess returns" which is the difference between portfolio return and T-Bill return, we want to explore the "excess return" between SP500 stocks returns and SP500 index.
We are particularly interested in how they volatile 20 days before and after the dividend record date.

Full Body Text

  • I used wrds CRSP - Daily Extract with Time Window tab to download each company's Trading Days Relative to Event Date(td_count), Price, Return on S&P Composite by uploading a txt file containing all SP500 companies' PERMNOs and dividend record dates
  • I groupby td_count, and used the Panda function pct_change() to compute each stock's return, and subtract the return on SP500 on to compute Excess Return

    • We observe the significant drop on the ex-dividend date (indexed as -1), which is one day before the record date (indexed as 0); this makes sense since the prices of the stock are marked down by the amount of dividends.
    On the record day the SP500's price also drops so the idiosyncratic effects are balanced out; the excess return bounces back around 0.

import pandas as pd
import numpy as np
import matplotlib.pyplot as plt
df = pd.read_csv("HW2.csv",sep=',')
spmean=df.groupby(['td_count'])['sprtrn'].mean()
prcmean=df.groupby(['td_count'])['PRC'].mean()
mean_workspace=spmean.to_frame()
prcmean=prcmean.to_frame()
mean_workspace["PRC"]=prcmean["PRC"]
mean_workspace["PRC_return"]=mean_workspace["PRC"].ffill().pct_change()
mean_workspace["Excess"]=mean_workspace["PRC_return"]-mean_workspace["sprtrn"]
_ = plt.plot(mean_workspace['Excess'])
_ = plt.title(r'Excess Rate of Return on SP500', fontsize=15)
plt.show()

HW 3 Annual Data Set

We would likely to work with large data set, for example, data manipulation across all SP500 companies' stock profiles. In this section of the project, I will create a dictionary consisting information we want.

Full Body Text

  • I uploaded a txt file of all SP500 companies' tickers to wrds CRSP - Stock Security File, and chose variables "Cusip", "Price", "Share Volume", "CRSP Permnant Company Number" to get the data
  • I computed the Market Capitalization by multiplying "Price" to "Share Volume"
  • I created a "dfDict" variable with all unique tickers, and use a loop to tell my original data frame to only generate variables that match that particular "Ticker"

    • All companies' profiles are stored in the dictionary ; to access we just print(dfDict['TICKER']). For example, let us access Apple's profile from year end 2012 to 2019 and compute their yearly compounded returns and add to the existing profile

    We observe the return went from positive to negative from 2012 to 2014 and moved upward from 2018 to 2019. We can do this analysis to any companies we want in the dictionary.
